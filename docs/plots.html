Chapter 1 Plots

All plots are drawn using the ggplot2 R-package. The default color scheme and theme are modified as below.

library(ggplot2)
theme_set(theme_classic(base_size = 12))
options(
  ggplot2.discrete.fill = list(
    RColorBrewer::brewer.pal(2, "Set1"),
    RColorBrewer::brewer.pal(8, "Set2")
  ),
  ggplot2.discrete.colour = list(
    RColorBrewer::brewer.pal(2, "Set1"),
    RColorBrewer::brewer.pal(8, "Dark2")
  )
)
## Warning in RColorBrewer::brewer.pal(2, "Set1"): minimal value for n is 3, returning requested palette with 3 different levels
